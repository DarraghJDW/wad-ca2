# Assignment 2 - Travel Destinations App

This is a web app that helps you keep track of destinations you've been to, you can record info such as
- Country
- Average travel cost
- Best season to visit
- A short description
- An image of the destination

The travel destinations are stored in a JSON collection which contains regions and destinations.  
The About page also loads statistics from a separate JSON file.

Features include
- User authentication (signup, login, logout)
- Each user manages their own collection of regions and destinations
- Add, edit and delete regions and destinations
- Image upload to Cloudinary for regions and destinations
- Search and sort regions on the dashboard
- App and user statistics on the About page
